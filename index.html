TopazChat 2.0

VRChat向けの、超低遅延ライブ配信システムです。OBSまたは専用アプリを使うことで、音声と映像を配信できます。VRChatワールドで視聴するために、AVProVideoを利用した専用のプレイヤーを配布しています。

下記の用途で利用されています。

  • NETDUETTOを利用した音楽セッションの共有
  • ゲーム画面をOBSで配信し、ワールド内で共有

採用実績

  • アルテマエンターテインメント株式会社
    アルテマナイト「年末振り返り配信！」2019/12/26

技術詳細

AWS EC2上にWowza Streaming Engineを設置し、VRChatワールド内のプレイヤーにRTSPで配信しています。専用の配信アプリTopazChat Streamerを使用することで、東京であれば約0.3秒の遅延で音声を配信することができます。

AudioTeapot

任意のWindowsアプリに内蔵されたノイズゲートを無効化するアプリです。VRChatでボイスチャットを介して音楽演奏する際に、VRChatのノイズゲートによって音が途切れがちになる現象を回避する用途を想定しています。

About

フリーランスのオーディオプログラマーです。株式会社CRI・ミドルウェアにてUnreal Engine 4のエディタ拡張プラグインの開発、株式会社エクシヴィにてアニメ制作のためのVRツール『AniCast』の開発に従事。

Unityでの開発経験は下記のとおりです。

  • Windows, Mac OS X, Android向けのネイティブプラグイン開発
  • エディタ拡張による開発効率化
  • VRアプリケーションのパフォーマンス最適化
  • アニメーション記録や、指・表情などキャラクター制御
  • 動画のガンマ補正など簡単なシェーダー記述
  • Googleのクラウドプラットフォーム（GCP）を利用した、アプリケーションのパフォーマンスモニタリング
  • Amazonのクラウドプラットフォーム（AWS）を利用した、超低遅延小規模ライブ配信の開発

療養中ではありますが、お仕事を受け付けております。現在は株式会社エクシヴィ、およびアルテマエンターテインメント株式会社と取引中。
